# Emoji Picker

A custom VS Code extension to insert emojis using shortcuts.

## Features

Type emoji names and choose from suggestions.

Examples:

:smile → 😀  
:fire → 🔥  
:bug → 🐛  
:rocket → 🚀  
:check → ✅

## Works In

- JavaScript
- Python
- Java
- C++
- HTML
- CSS
- Any text file

## How To Use

1. Type `:`
2. Start typing emoji name
3. Select emoji from suggestion list
